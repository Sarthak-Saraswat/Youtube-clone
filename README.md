## YouTube Clone
A simple YouTube-like UI built with Next.js and Tailwind CSS.

### Features
- Dark mode support
- Video thumbnails & search UI
- Responsive layout

### Tech Stack
- Next.js
- Tailwind CSS
- TypeScript

### Run Locally
```bash
git clone https://github.com/Sarthak-Saraswat/Youtube-clone.git
cd Youtube-clone
npm install
npm run dev
